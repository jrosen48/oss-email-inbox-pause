Generate App Icons

Instructions:

  1. Open this file in your browser
  2. Use browser screenshot tools or macOS Screenshot (Cmd+Shift+4) to capture each icon
  3. Save each as: icon-16.png, icon-32.png, icon-80.png, icon-128.png
  4. OR use the command below to auto-generate all sizes

Auto-generate using command line (requires ImageMagick):

brew install imagemagick
cd assets/icons
for size in 16 32 80 128; do convert icon.svg -resize ${size}x${size} icon-${size}.png; done
