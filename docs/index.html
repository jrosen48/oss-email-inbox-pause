Installation

The add-in is currently in review for Microsoft AppSource. In the meantime, you can:

  1. Deploy your own instance following the deployment guide
  2. Wait for AppSource approval (coming soon)

How It Works

  1. Click "Pause Inbox" in Outlook
  2. New emails are automatically moved to a "Paused" folder
  3. Click "Resume" when you're ready
  4. All paused emails return to your inbox

Privacy

We take privacy seriously:

  • No email content is read or stored
  • Only folder operations and metadata
  • Open source - verify the code yourself
  • Self-hostable for complete control
